PicoUrl

    pico <== Nano <== MICRO

How to:

1) Go to https://picoURL.herokuapp.com/new/<your-link-here>

2) PicoUrl responds with a JSON object containing your long link and your pico'd link

Examples:

Request: https://header-giver.herokuapp.com/new/www.google.com

Response: { original: "http://www.google.com",
short: "https://picoURL.herokoapp.com/sldfkjsldkjf" }
